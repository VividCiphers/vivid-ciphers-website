Skill set_

Go from Zero to Hero! And learn a variety of Skills along the Way

Our organisation has a lot of people experienced in Computer Science and IT. We keep learning and being up to date on skills allow us to work and teach you to code from scratch for as long as you're interested in learning programming. We never give up and we do everything at the best level, writing clean code you can understand and update. Along with having a play with algorithms, never stop learning and you'll be able to build awesome programms or websites. Want to build a unique UI interface with amazing UX? Not a problem! We'll help you on it! Want to create your own game? Perfect! We would be interested in creating one as well :)

We didn't stop learning at web development, we want to know more, do more. To be able to build next generation aniviruses, browsers and know how it all works overall. Our developers are learning (or have started to learn) software and game development. For gamedev we are using Unity, Unreal Engine and CryEngine as the platforms to build and test stuff on. We are learning new frameworks and practicing our skills by doing runs on front or backend daily basis. We use discord to communicate and discuss possible solutions, projects and much more. And if you want to try freelance - we are happy to help. Wanna work with us or hire us? Be welcomed to leave a message :)
